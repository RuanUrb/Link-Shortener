# My solution to Backend Brasil #1 challenge

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API endpoints

POST to / expects a {"url": "<url>"} body and returns as response your shortened URL

GET to /<shortenedUrl> will redirect you to original URL
